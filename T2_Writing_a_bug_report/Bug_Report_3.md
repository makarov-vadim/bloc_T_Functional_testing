# Баг репорт 3. API. Несоответствие кода успешного выполнения запроса

## Описание
В случае успешного выполнения запроса получен ответ, несоответствующий ожидаемому. 

### Ожидаемый результат
Ответ с кодом 200.
### Фактический результат
Ответ с кодом 201.
